# VSCode 使用心得

VSCode 是一個非常好用的編輯器, 但如果沒有好好的設定, 那麼它就只是一個普通的編輯器而已, 所以我們要來打造一個屬於自己的 VSCode

## 安裝軟體

首先我們要先安裝 VSCode, 這個就不多說了, 直接去官網下載, 或是使用 `brew` 安裝

```bash
brew install --cask visual-studio-code
```

## 擴充套件

VSCode 有非常多的擴充套件, 我們可以透過擴充套件來打造一個屬於自己的 VSCode, 以下是我常用的擴充套件

### Vibrancy Continued - 透明化主題

可以讓 VSCode 變成透明主題, 好看、真的好看

### Tokyo Night - 主題

我使用的主題, 可以搭配 Vibrancy Continued 來使用

### Power Mode - 打字特效

打字特效、Combo, 可以讓你的 VSCode 變得更有趣

### Live Share - 協同編輯

可以讓你和你的夥伴一起編輯同一個檔案, 非常方便

### GitLens - Git 工具

可以讓你更方便的使用 Git

### GitHub Copliot - AI 程式碼建議

可以讓你更方便的寫程式碼

### CodeSnap - 程式碼截圖

可以讓你更方便的截圖
